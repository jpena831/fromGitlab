This semester, I was a part of the course "Code for Her" at Penn State University. Throughout this couse, we learned the basics of html, CSS, and JavaScript. The purpose of creating this course - thank you Carmen Cole - was to allow a space in which females, and gender-diverse students could learn to code.

Welcome! On my page you will find some things that I have done throughout the Code for Her Fall 2017 Course!
